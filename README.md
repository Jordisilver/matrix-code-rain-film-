# 🏛️ Matrix Rain v2.1 | Architect Edition

The ultimate Matrix digital rain emulation. Version 2.1 "Architect" is a complete rewrite using Object-Oriented Programming (OOP) to deliver professional-grade visuals and a built-in media player.

## 🚀 Evolution: What's New in v2.1?

| Feature | v1.4 | **v2.1 Architect (Final)** |
| :--- | :--- | :--- |
| **Rendering** | Basic Surface blit | **True Alpha-Blending** (Smooth tails) |
| **Audio** | Single track | **Full Playlist Support** (Folder-based) |
| **Media Player** | Auto-play | **Interactive Controls** (Pause / Next) |
| **Customization** | Speed/Size | **Tail Length & Mutation Control** |
| **Architecture** | Functional | **Class-based (OOP)** |
| **Quick Launch** | Terminal only | **One-click .bat launch** |

## 🕹️ Interactive HUD 3.0
The hidden settings panel (top-right corner) now includes:
* **[SIZE]** — Dynamic font scaling.
* **[SPEED]** — Flow intensity control.
* **[LEN]** — Tail length adjustment.
* **[MODE]** — Switch between Katakana, Binary, Hex, and Mixed.
* **[PLAYER]** — Fully functional music controller for your `.mp3` library.

## 📂 Installation & Setup
1. **Prepare Music:** Create a folder named `music` next to the script and drop your favorite `.mp3` files there.
2. **Launch:** * **Windows:** Simply double-click `matrix_v2.1_architect.bat`.
   * **Other:** Run `python matrix_v2.1_architect.py`.

## ⌨️ Global Hotkeys
* **[C]** — Cycle color schemes (Green, Red, Blue, Gold, White).
* **[ESC]** — Safe system exit.
