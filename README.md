# 🍏 Matrix Rain v1.2 | Interactive Fullscreen Engine

A high-performance, interactive "Matrix Digital Rain" emulation built with Python and Pygame. This version (v1.2) is a significant upgrade, featuring an immersive fullscreen experience, dynamic UI, and real-time user interaction.

## 🌟 Key Features (v1.2)
* **Smart Fullscreen:** Automatically detects your display resolution for a seamless experience.
* **Interactive HUD:** A hidden settings panel that appears when you hover your mouse in the top-right corner.
* **Multi-Color Core:** Switch between 5 iconic color schemes (Green, Red, Blue, Gold, White) instantly.
* **Dynamic Speed:** Adjust the "flow" of the Matrix in real-time using keyboard controls.
* **Authentic Characters:** High-quality rendering of Japanese Katakana and numeric sets.

## 🎮 Controls & Hotkeys
* **[C]** — Cycle through available colors.
* **[UP Arrow]** — Increase rain speed (Max 120 FPS).
* **[DOWN Arrow]** — Decrease rain speed (Min 5 FPS).
* **[ESC]** — Exit the application.
* **[Mouse Hover]** — Reveal the Settings Panel (Top-Right corner).

## 🛠 Technical Stack
* **Language:** Python 3.12+ (Recommended)
* **Library:** Pygame 2.5.2+
* **Font:** MS Gothic / Consolas (Optimized for bold rendering).
