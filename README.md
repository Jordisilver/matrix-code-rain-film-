# 🍏 Matrix Rain v1.2 vs v1.4 | Architect Edition

An advanced, highly interactive Matrix digital rain engine built with Python. This version is a massive upgrade from v1.2, focusing on visual fidelity and deep user control.

---

## 🚀 Key Improvements (v1.2 vs v1.4)

| Feature | Version 1.2 | **Version 1.4 (Current)** |
| :--- | :--- | :--- |
| **Visuals** | Static falling characters | **Dynamic Glitch Effect** (mutates mid-air) |
| **Glow** | Single color columns | **Flash Highlights** (random bright spikes) |
| **HUD / UI** | Hover-only info panel | **Interactive HUD 2.0** (clickable buttons) |
| **Data Modes** | Mixed only | **Binary, Hex, Katakana, and Mixed** |
| **Audio** | No audio | **Integrated Soundtrack Support** (`.mp3`) |
| **Physics** | Uniform speed | **Variable Velocity** (each column is unique) |

---

## 🌟 New Visual Logic
* **Glitch Effect:** Characters have a 10% chance to change while falling, creating a "live stream" feel.
* **Smart HUD:** You can now change **Font Size** and **Data Modes** directly with your mouse.
* **Audio Engine:** Background music synchronization for an immersive experience.

---

## 🎮 Controls
* **[Mouse Hover]** — Show Settings Panel (Top-Right).
* **[Mouse Click]** — Adjust Size, Speed, and Modes on the HUD.
* **[C] Key** — Cycle through 5 Cinematic Colors.
* **[UP/DOWN]** — Global FPS Control.
* **[ESC]** — Exit System.
