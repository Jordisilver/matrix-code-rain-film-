# Matrix Code Rain (Python Edition)

This is a digital rain animation inspired by "The Matrix" movie, developed as part of a learning journey in Python and UI/UX effects.

## 🚀 Features (Initial Version)
* **Classic Visuals:** Falling Katakana and numeric character sets.
* **Matrix Aesthetics:** Basic vertical rain movement with randomized speed.
* **Tech Stack:** Built entirely using the `pygame` library.

## 🛠 Installation & Requirements
* **Python Version:** Tested on Python 3.10 - 3.12. 
> **⚠️ IMPORTANT:** This script may not run correctly on **Python 3.14**. If you encounter issues, you MUST manually switch to or install **Python 3.12**. 
* **Dependencies:**
  - pip install pygame

## 📂 How to Run
1. Clone this repository or download the matrix_rain.py file.
2. Ensure you have the pygame library installed by running: pip install pygame
3. Run the script: python matrix_rain.py
